# Vozes do Direito

Site desenvolvido para o projeto de extensão **Vozes do Direito**, com o objetivo de aproximar o Direito da sociedade por meio de conteúdos jurídicos em linguagem simples, acessível e educativa.

## Objetivo

Democratizar o acesso à informação jurídica, traduzindo termos, conceitos e situações do cotidiano para uma linguagem clara e compreensível.

## Tecnologias utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Git e GitHub

## Estrutura do site

- Página inicial
- Conteúdos
- Dicionário Jurídico
- Direito no Cotidiano
- Quem Somos
- Contato
